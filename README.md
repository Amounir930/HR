# Mobica HR & Payroll Suite

A comprehensive Human Resources and Payroll management system tailored for the Egyptian market (Laws 91/2005, 148/2019, 14/2025).

## Features

- **Dashboard**: Real-time insights into headcount, attendance, and payroll.
- **Employee Management**: Full employee records, positions, and departments.
- **Egyptian Payroll Engine**: 
  - Progressive tax calculation (2026 brackets).
  - Social Insurance (Law 148/2019).
  - Universal Health Insurance.
  - Law 14/2025 Labor Law compliance.
- **Attendance & Leave**: Track presence, late entries, and leave entitlements.
- **Performance**: Goals (OKRs) and performance reviews.
- **Recruitment**: Job openings and candidate tracking.
- **Training**: Course catalog and enrollment management.
- **Compliance & Safety**: Disciplinary actions and incident reporting (Near Miss, Injury, etc.).

## Tech Stack

- **Framework**: React 19 + Vite 8
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Storage**: LocalStorage (Persistent across sessions)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run in development:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
